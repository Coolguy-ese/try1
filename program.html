FLEX-BOX

A Flex box also known as flexible boxes provides a more efficient way to layout, align and distribute spaceamong items in acontainer. Flex box specify how flex items are set inside the Flex container.

PROPERTIES OF A FLEX BOX

  1. Flex-direction:

    The flex directiom is used to specify the direction of a flexible item in a flex container. This property can take the following value:
    • Column:
    • Column-reverse
    • row-reverse
    • Row


  2. Justify-content:

    It is used to align the flex item horizontally when the itemcould not use up avaliable space on the main axis Value:
    • Flex-end
    • Flex-start
    • Center
    • Space-between


  3. Flex-wrap:

    This properly is used to specify whether the flex item should be wrapped or not when there is no enough room for one flex line Value:
    • Wrap
    • Roomwrap
    • Wrap-reverse
